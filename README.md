# Freelance Timer

A minimal macOS menubar app to track freelance work sessions with projects, summaries, and local-only storage (no backend).

## Features
- Menubar stopwatch with Start/Pause/Resume/Finish
- Projects & companies with color tags
- Manual session entry
- Day/Week/Month/Year range summaries + chart
- Onboarding for first-time setup
- Local Core Data storage (offline-first)
- Export sessions to CSV
- Reset all data from Settings

## Requirements
- macOS 13+
- Xcode 15+

## Run
1. Open `Freelance Timer.xcodeproj` in Xcode.
2. Select **My Mac** as the destination.
3. Build & Run.

## Data Reset / Export
In the app: **Settings → Export CSV…** or **Reset All Data…**

## Notes
- Menubar shows only active projects. Manage projects in the main app window.
- App icon assets live in `Freelance Timer/Assets.xcassets/AppIcon.appiconset`.
